# shopsphere-api

A modern and scalable e-com REST API built using FastAPI, PostgreSQL, and Stripe

## Features

- JWT Authentication & Authorization
- Product Catalog Management
- Shopping Cart Function
- Stripe Payment
- Order Management System
- Redis Caching
- Testing
- CI/CD Pipeline

## Tech Stack

- **FRWK** FastAPI
- **Database** PostgreSQL
- **Cache** Redis
- **Authentication** JWT
- **Payment** Stripe
- **Testing** Pytest
- **Deployment** Docker

## Prereq

- Python 3.9+
- PostgreSQL
- Redis
- Docker
